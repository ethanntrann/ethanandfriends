# ethanandfriends

A simple senior trip planner for Ethan, Angela, Doria, and Justin.

The app helps the group:

- Compare cheap flight search tools for summer 2027.
- Find hotel search links for the selected city or country.
- Discover popular activities and tours.
- Add and remove shared itinerary events in a browser-saved calendar.

## Deploying on Render

This repo includes `render.yaml` for a Render static site.

1. Push this repository to GitHub.
2. Open Render and create a new Blueprint from the GitHub repo.
3. Render will use `render.yaml` and serve the site as a static app.

No API key is required for the current version. Keep any future real travel or AI API keys in Render environment variables, not in the frontend code.
